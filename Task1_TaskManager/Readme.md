# Number Guessing Game

## 📌 Description
The Number Guessing Game is a console-based C++ application developed as part of the **CodSoft C++ Programming Internship**. The program generates a random number, and the user has to guess it. The game provides hints such as "Too High" or "Too Low" until the correct number is guessed.

## 🚀 Features
- 🎲 Random number generation
- 🎯 Three difficulty levels:
  - Easy (1–50)
  - Medium (1–100)
  - Hard (1–200)
- 📈 Counts the number of attempts
- 💬 Provides hints (Too High / Too Low)
- 🏆 Displays performance based on attempts
- 📋 User-friendly console interface

## 🛠️ Technologies Used

- Functions
- Loops
- Conditional Statements
- `<cstdlib>` Library
- `<ctime>` Library
