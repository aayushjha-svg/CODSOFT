# Library Management System

## 📌 Description
The Library Management System is a console-based C++ application developed as part of the **CodSoft C++ Programming Internship**. It helps manage library books by allowing users to add, search, issue, return, and delete books. The system also stores book records permanently using file handling.

## 🚀 Features
- 📚 Add New Books
- 📖 View All Books
- 🔍 Search Books
  - By Title
  - By Author
  - By ISBN
- 📤 Issue Books
- 📥 Return Books
- ❌ Delete Books
- 📊 Dashboard
  - Total Books
  - Available Books
  - Issued Books
- 💾 File Handling (`books.txt`)
- ✅ Duplicate ISBN Validation
- 📋 Menu-Driven Interface

## 🛠️ Technologies Used
- C++
- Object-Oriented Programming (OOP)
- STL Vector
- File Handling
- Functions
- String Handling
