# Smart Calculator

## 📌 Description
The Smart Calculator is a menu-driven console application developed in **C++** as part of the **CodSoft C++ Programming Internship**. It performs basic as well as advanced mathematical operations through an interactive user interface.

## 🚀 Features
- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division (with divide-by-zero validation)
- 🧮 Modulus Operation
- 🔢 Power Calculation
- √ Square Root Calculation
- 📋 Menu-Driven Interface
- ❌ Invalid Input Handling

## 🛠️ Technologies Used
- C++
- Functions
- Switch Case
- Loops
- `<cmath>` Library

## ▶️ How to Run

### Compile

```bash
g++ main.cpp -o calculator
```

### Run

**Windows**
```bash
calculator.exe
```

**Linux/macOS**
```bash
./calculator
```
